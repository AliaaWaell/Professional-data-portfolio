# Sales Performance Dashboard

## Project Overview
This project demonstrates the use of Microsoft Excel to analyze sales data and build an interactive dashboard that summarizes key business performance indicators.

## Objective
To transform raw sales data into meaningful insights using Pivot Tables, Pivot Charts, and dashboard visualization techniques.

## Tools Used
- Microsoft Excel

## Skills Demonstrated
- Data Analysis
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization
- Reporting
- Business Insights

## Dashboard Components
- Total Sales
- Sales by Region
- Sales by Category
- Top 10 Products
- Orders by Segment
- Monthly Sales Trend

## Workbook Structure
- Raw Data
- Sales by Region
- Sales by Category
- Top 10 Products
- Orders by Segment
- Monthly Sales
- Dashboard

## Files Included
- Sales-Performance-Dashboard.xlsx
- Dashboard-Report.pdf

## Result
Built an interactive Excel dashboard that summarizes sales performance and supports business decision-making through visual reports.
