# Technology Companies Research Database

## Project Overview
This project demonstrates web research, data collection, and data organization skills by building a structured database of technology companies using publicly available information from official websites and LinkedIn.

## Objective
To collect, verify, classify, and organize company information into a structured Excel database suitable for business research and recruitment purposes.

## Tools Used
- Microsoft Excel
- LinkedIn
- Official Company Websites

## Skills Demonstrated
- Web Research
- Data Collection
- Data Entry
- Data Classification
- Data Verification
- Microsoft Excel
- Attention to Detail

## Dataset
The database contains information for more than 70 technology companies.

The collected information includes:
- Company Name
- Location
- Website
- Company Type
- LinkedIn Profile
- Industry
- Careers Page

## Files Included
- Technology Companies Database.xlsx

## Key Outcome
Built a structured and verified database that can support job searching, recruitment research, and market analysis while maintaining consistent formatting and accurate publicly available information.
